# Pizza Ecommerce React 🍕

## Descripción

Aplicación web desarrollada con React y Vite que simula una plataforma de venta de pizzas online. El proyecto incorpora autenticación de usuarios, gestión global de estado mediante Context API, navegación con React Router, consumo de APIs REST y funcionalidades de carrito de compras.

## Funcionalidades

- Registro de usuarios.
- Inicio y cierre de sesión.
- Persistencia de sesión mediante LocalStorage.
- Visualización de catálogo de pizzas.
- Detalle individual de productos.
- Carrito de compras.
- Incremento y disminución de cantidades.
- Cálculo automático del total.
- Checkout para usuarios autenticados.
- Navegación dinámica mediante React Router.
- Página 404 para rutas inexistentes.

## Tecnologías utilizadas

- React 19
- Vite
- React Router DOM
- Context API
- JavaScript (ES6+)
- Bootstrap
- CSS3
- Fetch API
- LocalStorage

## Conceptos aplicados

- Componentes reutilizables.
- Hooks de React.
- Estado global con Context API.
- Renderizado condicional.
- Consumo de APIs REST.
- Autenticación basada en tokens.
- Persistencia de datos.
- Gestión de rutas dinámicas.

## Instalación

Clona el repositorio:

```bash
git clone URL_DEL_REPOSITORIO
```

Instala las dependencias:

```bash
npm install
```

Inicia el proyecto:

```bash
npm run dev
```
